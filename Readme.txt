README : FLATHTML/800EXAMPLES
=============================

1. The main objective of this project is to develop ready to use code that one frequently uses for Web Development.

2. The code covers HTML / CSS /JavaScript JSON / AJAX / JQUERY /PHP /MySQL /XML / XSLT

3. Meta data of the code can be found at the file 800-Examples-list.xlsx 

4. If you want to use any piece of code , please download all the files related to the number. For instance if you
   want to use 047-appendxml.php , then make sure you have downloaded 047-append.xsl and 047-append.xml and placed
   in the same folder.

5. The code has been tested using the Apache and MySql server from Xampp. 

6. Hope you find this project useful. Any questions, please drop an email at d_menashi@yahoo.com
