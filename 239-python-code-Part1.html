EXAMPLE-1 : Read from XML file 
 from xml.dom import minidom
 
 
# parse an xml file by name
file = minidom.parse('people.xml')

#use getElementsByTagName() to get tag
daya1 = file.getElementsByTagName('name')

# all items data
print('\nAll name data:')
for elem1 in daya1:
    print(elem1.firstChild.data)

    #use getElementsByTagName() to get tag
daya2 = file.getElementsByTagName('person')

# all items data
print('\nAll person data:')
for elem2 in daya2:
    print(elem2.attributes['id'].value)
The above program reads data from XML file and just displays it
EXAMPLE-2 : Read from XML file using pandas and display data in CSV format 
 import pandas as pd
import xml.etree.ElementTree as et
print('begin')

xtree = et.parse("students.xml")
xroot = xtree.getroot()

df_cols = ["name", "email", "grade", "age"]
rows = []

for node in xroot:
    s_name = node.attrib.get("name")
    s_mail = node.find("email").text if node is not None else None
    s_grade = node.find("grade").text if node is not None else None
    s_age = node.find("age").text if node is not None else None

    rows.append({"name": s_name, "email": s_mail,
                 "grade": s_grade, "age": s_age})

out_df = pd.DataFrame(rows, columns = df_cols)
print (out_df.values)

print('its done')
The above program displays the contents of a XML file and displays in CSV format.
